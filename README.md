# MediaCenter
智能网络机顶盒媒体中心

媒体中心是一个集内部存储、SD卡、U盘、移动硬盘、Samba存储、NFS存储、DLNA服务等设备的扫描发现，文件浏览，视频播放，音乐播放，图片浏览，APK安装于一体的Box TV应用，支持Android 4.4-7.1平台。
